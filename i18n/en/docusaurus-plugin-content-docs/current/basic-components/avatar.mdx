# air-avatar
## Attributes
| Attribute name   | Type                             | Description                                         | Default value      |
| -------- | -------------------------------- | -------------------------------------------- | ----------- |
| `name`   | `string`                         | User name, used to display initials.                          | `''`        |
| `src`    | `string`                         | The URL of the profile picture.                                | `''`        |
| `size`   | `string`                         | The size of the profile picture. Valid values: `small`, `medium`, `large` or custom. | `'medium'`  |
| `shape`  | `'circle' \| 'square'`           | The shape of the avatar, supported are `circle` or `square`.                 | `'circle'`  |
| `border` | `string`                         | Custom border style.                                     | `''`        |
---
