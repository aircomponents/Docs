# air-icon

`air-icon` 是一个通用的图标组件，支持集成多种图标库（如 Material Icons、Font Awesome、Boxicons、Iconfont、IconPark 等），让图标的使用更加简单统一。

---

## 属性说明

| 属性名     | 类型     | 描述                                                                                           | 默认值    |
| ---------- | -------- | ---------------------------------------------------------------------------------------------- | --------- |
| `icon-set` | `string` | 指定使用的图标库前缀，例如 `material-icons`、`fas`、`bx`、`iconfont`、`iconpark` 等。 | 无         |
| `name`     | `string` | 图标名称，与对应图标库的图标名称一致。                                                           | 无         |
| `size`     | `string` | 图标尺寸，支持任意有效的 CSS 尺寸值（如 `2rem`、`24px`）。                                       | `'1rem'`  |
| `color`    | `string` | 图标颜色，支持任意有效的 CSS 颜色值（如 `red`、`#333`）。                                        | `inherit` |

---

## 示例
