# air-avatar

## 属性

| 属性名   | 类型                             | 描述                                         | 默认值      |
| -------- | -------------------------------- | -------------------------------------------- | ----------- |
| `name`   | `string`                         | 用户姓名，用于显示首字母缩写。                          | `''`        |
| `src`    | `string`                         | 头像图片的 URL 地址。                                | `''`        |
| `size`   | `string`                         | 头像尺寸，支持 `small`、`medium`、`large` 或自定义值。 | `'medium'`  |
| `shape`  | `"circle" \| "square"`           | 头像形状，支持 `circle` 或 `square`。                 | `'circle'`  |
| `border` | `string`                         | 自定义边框样式。                                     | `''`        |

---

### 1. 自定义头像图片的示例

<air-previewer size="auto">
<div style={{ display: 'flex', gap: '1rem', justifyContent: 'center' }}>
  <air-avatar
    size="large"
    name="李四"
    shape="square"
    src="https://avatars.githubusercontent.com/u/146103794?v=4"
    class="air-bg-secondary"
  ></air-avatar>
</div>
</air-previewer>
