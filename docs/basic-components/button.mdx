### Example Usage
### 1.  Basic buttons (default state)


Shows a minimalist button with an appearance that can be adjusted based on `size`, `variant` and `color`.

```jsx live
<air-button size="medium" variant="solid" color="primary">
  Primary Button
</air-button>
```

### 2. Buttons of different colours

Displays buttons in different colours, covering `primary`, `success`, `info`, `warning`, `danger`, and `ghost`.

```jsx live
<>
<air-button size="medium" variant="solid" color="success">Success Button</air-button>
<air-button size="medium" variant="solid" color="info">Info Button</air-button>
<air-button size="medium" variant="solid" color="warning">Warning Button</air-button>
<air-button size="medium" variant="solid" color="danger">Danger Button</air-button>
<air-button size="medium" variant="solid" color="ghost">Ghost Button</air-button>
<air-button size="medium" variant="solid" color="primary">Primary Button</air-button>
</>
```

### 3. Demonstration of buttons of different sizes 
The `size` attribute has different effects. You can use `small`, `medium`, and `large` to display different button sizes.

```jsx live
<>
<air-button size="large" variant="solid" color="primary">Large Button</air-button>
</>
```

### 4. Buttons with icons
Displays buttons with icons. Use the `icon` attribute to add prefix and suffix icons to the buttons.

```jsx live
<>
<air-button size="medium" variant="solid" color="primary" icon="ðŸ“¥">Download Button</air-button>
</>
```

### 5. Loading status button
Use the `loading` attribute to show the button while it is loading.

```jsx live
<air-button size="medium" variant="solid" color="primary" loading>Button</air-button>
```

### 6. Disable status button
To display buttons in a disabled state, use the `disabled` attribute.

```jsx live
<air-button size="medium" variant="solid" color="primary" disabled>Disabled Button</air-button>
```

### 7. Select the status button
Show the button in the selected state, using the `selected` property.

```jsx live
<air-button size="medium" variant="solid" color="primary" selected>Button</air-button>
```

### 8. Buttons with slot contents
Display buttons with slotted content (e.g. text).

```jsx live
<air-button size="medium" variant="solid" color="primary">
  <span>Slot Button</span>
</air-button>
```

### 9.Combination example: buttons with different properties
Showcase multiple combinations of attributes to highlight the state and appearance of different buttons.

```jsx live
<>
<air-button size="large" variant="outline" color="success" icon="ðŸ”" loading>Mix Button</air-button>
</>
```
